- FastAPI

# Step-by-step

## Prerequisites

- python + pip

## on windows: virtual environment

### install virtualenv

```sh
    pip install virtualenv
```

### start virtualenv

```sh
    env/Scripts/activate
```

### in case of "UnauthorizedAccess" or "PSSecurityException"

```sh
    Set-ExecutionPolicy Unrestricted -Scope Process
```
